Descripción del Proyecto

De forma general, este proyecto de prácticas se basa en la creación de una aplicación de escritorio sobre datos del mundo del fútbol, relacionados tanto con el videojuego FIFA como partidos, historia, hechos históricos u otros.

El entorno de desarrollo ha sido Visual Studio 2022, utilizando XAML como Front-End y C# como Back-End
