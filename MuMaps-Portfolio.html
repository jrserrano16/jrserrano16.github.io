Descripción del Proyecto

De forma general, este proyecto de prácticas se basa en la creación de una aplicación que genere un Playlist de música con una duración similar al viaje seleccionado. Para ello combinamos 2 Apis: Spotify y Here

El entorno de desarrollo ha sido Visual Studio 2022, utilizando XAML como Front-End y C# como Back-End
